# Portfolio

This is a Next.js 16 portfolio website built with the App Router.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Features

- Next.js 16 App Router
- React 19
- Responsive portfolio layout
- Custom global styling
- Contact form integration

## Notes

- `resend` is installed for email handling.
- `.gitignore` excludes Node, Next.js build output, environment files, and editor caches.
